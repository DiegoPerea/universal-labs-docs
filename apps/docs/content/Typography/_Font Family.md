# Font Family

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class    | Native | Web |
| -------- | ------ | --- |
| font-{t}    | ✅     | ✅  |
| font-{a}    | ✅     | ✅  |
| font-sans   | ❌     | ✅  |
| font-serif    | ❌     | ✅  |
| font-mono    | ❌     | ✅  |
