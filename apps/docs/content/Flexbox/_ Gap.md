# Gap

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values 

| Class     | Native | Web |
| --------- | ------ | --- |
| gap-{t}   | ❌     | ✅  |
| gap-{a}   | ❌     | ✅  |
| gap-x-{t} | ❌     | ✅  |
| gap-x-{a} | ❌     | ✅  |
| gap-y-{t} | ❌     | ✅  |
| gap-y-{a} | ❌     | ✅  |
